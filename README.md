# PCA Face Recognition Midterm Project

This repository contains the Python implementation and LaTeX report for a Face Recognition system utilizing Principal Component Analysis (PCA)

## Dataset
This project uses the **ORL Database of Faces** (`attface.zip`). It contains 400 grayscale images of 40 different subjects.

## Project Structure 
* `main.py` : The Python script that unzips the data, applies PCA to extract 50 eigenfaces, and calculates Euclidean distances for facial recognition.
* `main.tex` : The LaTeX source code for the mathematical report.
* `SOC_midterm_report.pdf` : The final compiled midterm report.
